## 📥 How to Download Images

### Step 1: Find the Image
1. Go to brand website (links above)
2. Look for campaign/collection sections
3. Right-click on image
4. Click "Save image as..."
5. Choose your Downloads folder

### Step 2: Prepare the Image
1. Open image in any image editor (Paint, Photoshop, GIMP)
2. Check size: ideally 400-600px width
3. Optional: Crop to poster aspect ratio (1:1.35)
4. Save as JPG (quality 85%)

### Step 3: Optimize (Optional but Recommended)
- Use online tool: https://tinyjpg.com
- Compress to 150-300KB
- Keep resolution 400-600px width

### Step 4: Place in Correct Folder
1. Copy optimized JPG
2. Go to: `c:\Veyronnnnnnnnnn\public\posters\[category]\`
3. Paste file
4. Delete old `.html` placeholder file

---
