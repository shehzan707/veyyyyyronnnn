## ❓ TROUBLESHOOTING

### Filter button not showing?
- Make sure page is at http://localhost:8000/products
- Check browser console (F12) for errors

### Categories not loading?
- Click a gender option first
- Wait 1 second for categories to load

### Products not updating?
- Make sure you're checking checkboxes (not just viewing)
- Check browser console for network errors

### Price slider not working?
- Drag the circular handles (not the track)
- Or type numbers in Min/Max fields
