## 🎯 Theme Toggle Button

### Appearance

```

│  ADMIN INTERFACE   │
│                    │

```

### Button Details

| Property | Value |
|----------|-------|
| **Shape** | Perfect Circle (44×44px) |
| **Position** | Fixed, Bottom-Left |
| **Offset** | 16px from left, 16px from bottom |
| **Z-Index** | 1000 (always on top) |
| **Default Opacity** | 0.5 (subtle, non-intrusive) |
| **Hover Opacity** | 1.0 (fully visible) |
| **Background** | Matches theme (card-bg) |
| **Border** | Subtle, theme-aware |
| **Transition** | 250ms smooth fade |
| **Icons** | Material Icons (light_mode / dark_mode) |

### Button States

**Light Theme:**
```
Normal:  ☀️ (50% opacity, light grey circle)
Hover:   ☀️ (100% opacity, white circle with shadow)
Active:  🌙 (starts showing moon icon, fading in)
```

**Dark Theme:**
```
Normal:  🌙 (50% opacity, dark grey circle)
Hover:   🌙 (100% opacity, darker circle with shadow)
Active:  ☀️ (starts showing sun icon, fading in)
```

## 🎨 CSS Variables in Action

### How it works internally:

```css
/* Light Theme - Applied when body.theme-light */
body.theme-light {
    --bg-primary: #ffffff;      /* Page background */
    --text-primary: #000000;    /* Main text */
    --btn-bg: #333333;          /* Button background */
    --card-bg: #ffffff;         /* Card backgrounds */
}

/* Dark Theme - Applied when body.theme-dark */
body.theme-dark {
    --bg-primary: #2a2a2a;      /* Page background */
    --text-primary: #ffffff;    /* Main text */
    --btn-bg: #ffffff;          /* Button background */
    --card-bg: #323232;         /* Card backgrounds */
}

/* All elements use these variables */
.element {
    background: var(--bg-primary);
    color: var(--text-primary);
    border: 1px solid var(--border-color);
}
```

When the theme changes, the CSS variables automatically update, and all elements transition smoothly to the new colors.

---

## 🔄 Smooth Transitions

All color changes animate over **250 milliseconds** using `ease` timing:

```css
body {
    transition: 
        background-color 250ms ease,
        color 250ms ease;
}

.header {
    transition: 
        background-color 250ms ease,
        color 250ms ease,
        border-color 250ms ease,
        box-shadow 250ms ease;
}
```

This provides a smooth, non-jarring visual experience.

## 💡 Design Philosophy

✅ **Clarity**: Text is always clearly readable
✅ **Consistency**: Same layout, different colors
✅ **Accessibility**: Maintains WCAG AA/AAA contrast
✅ **Subtlety**: No neon, bright, or flashy colors
✅ **Professionalism**: Luxury, refined appearance
✅ **Comfort**: Long-session eye comfort (especially dark theme)
✅ **Transitions**: Smooth, non-jarring color changes
